dotfiles
=======

Requirements:
  * Ruby
  * rake

Contains:
    
  * Ruby (rbenv)
  * Python
  * ZSH
  * Vim
  * Powerline

See [Brewfile](config/Brewfile) for all installed apps

Usage:

    git clone git@github.com:deiga/dotfiles.git
    cd dotfiles
    git submodules init
    git submodules update
    rake install
